Site Features

June 16, 2020 by David

Holy guacamole! You should check out some of this (mostly static) site's features below.
I've designed this blog according to the instructions outlined by Brian Yu for the edX HarvardX CS50W Web Programming with JavaScript and Python course. While some links and buttons may not be functional at the moment (I will activate later in the future), listed below are some of the features I've used so far in designing the website:
  • An index page which makes use of some of Bootstrap 4.5.0's (blog template) features.
  • Three other pages (About, Contact and Gallery) pages all accessible from the static navigation bar and accessible from each other.
  • An .scss file that maps to a corresponding .css file containing various selectors for styling specific sections (like this one for example).
  • @media queries for adding cross-device responsiveness.
  • HTML5 elements e.g links, buttons, forms, ordered and unordered lists, etc.
