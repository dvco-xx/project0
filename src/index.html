Site Features

June 16, 2020 by David

I've designed this site according to the instructions outlined by Brian Yu for the edX Harvard CS50 Introduction to JavaScript and Python course. While some links and buttons may not be functional at the moment (for which I deeply apologize and promise to fix later), listed below are some of the features I've used:
  • Bootstrap 4.5.0 blog template for the general site layout.
  • A sass file that maps to a corresponding css file containing various selectors for styling specific sections (like this one for example) and @media queries for adding cross-platform responsiveness.
  • HTML5 elements e.g links, tables, forms, ordered and unordered lists, etc.
